# ViralBoost Tools

A free React + Vite + Tailwind CSS website for creators who need quick social media content ideas.

## Features

- Facebook engagement post generator
- Caption generator
- Hashtag generator
- Thumbnail text generator
- TikTok hook generator
- Random preset generation with copy-to-clipboard
- One-page responsive dark creator-style design
- No backend, database, login, or AI API

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
